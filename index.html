Il existe 3 types d'actions :

  • Allumer / eteindre la lumiere
  • Changer la couleurs des lamps
  • Des actions speciales

Allumer / Eteindre la lumiere

Il suffit pour cela de dire : Sarah, lumiere.
Si la lumiere est eteinte, Sarah allumera la lumiere, et vice-versa.
Il est aussi possible d'utiliser :
- Sarah, allume la lumiere
- Sarah, eteins la lumiere

Changer la couleur

Il faut utiliser le mot-cle : Change la couleur
Les couleurs possibles sont :
  • Blanc
  • Bleu
  • Rouge
  • Jaune
  • Orange

Il faut preciser :
  • Toutes les lampes, pour changer toutes les lampes d'un coup
  • une seule lampe, si vous souhaitex en changer qu'une.
  • vous pouvez aussi precisez, le bloom ou le strip, si vous en avez.
  • un groupe (v0.3)

Ex :
  • Sarah, change la couleur de toutes les lampes en bleu
  • Sarah, change la couleur de la lampe de la cuisine en jaune.
  • Sarah, allume le groupe deux

De meme pour les groupes : philipshue.js
var groups=[["1","1","2","5"] , ["2","3","4"] ];

Ainsi que dans le fichier philipshue.xml
	
		
		du groupe un out.action.group="1";

Dans notre exemple, le groupe 1 regroupe les lampes 1, 2 et 5. La commande, "Sarah, eteins le groupe un", va donc eteindre la luminere du lit, de la cuisine et le strip.
